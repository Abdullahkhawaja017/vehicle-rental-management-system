# Car Rental Management System

## Overview
The Car Rental Management System is a desktop application designed to manage vehicle rentals efficiently. It supports customer registration, vehicle management, booking, billing, and return processing.

Built using Python (Tkinter) and Microsoft SQL Server, it provides a simple and user-friendly interface for managing rental operations.

## Features
- Customer management
- Vehicle inventory management
- Booking and reservation system
- Rental and return processing
- Billing and transaction records
- Admin control panel
- SQL Server database integration

## Technologies Used
- Python
- Tkinter
- Microsoft SQL Server
- SQL
